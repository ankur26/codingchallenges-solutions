# ccwc
## Requirements
BunJS needs to be installed

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run prog [OPTIONS] [FILE]
```

This project was created using `bun init` in bun v1.0.4. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
